# final_project_nanum_homepage (나눔 홈페이지)
행정안전부에서 제공하는 공공 API를 활용해 봉사활동을 원하는 봉사자에게 전국의 봉사활동정보 제공과
봉사 신청을 통한 봉사활동을 관할하는 기관간의 매칭을 해주고 게시판과 봉사후기를 제공해 커뮤니티를 활성화해
봉사에대한 흥미와 참여를 이끌어내고자하는 홈페이지
